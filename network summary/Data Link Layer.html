전반적인 역할

  1. 프레임 형성
    :이진 신호에 최초로 의미를 부여합니다.
  2. 물리 주소 부여
    :통신 제품에 부여되어 나오는 주소로, 각각의 통신매체를 구분합니다.
  3. FCS
    :이진 신호 전송에서 오류를 감지합니다.
  4. 충돌 도메인 분리
    :2계층 장비는 Port별로 충돌 도메인을 분리하여 충돌의 확장을 차단합니다.

세부목록

Bridge(브릿지)

소프트웨어를 기반으로 하며 Half-Duplex의 전송 방식을 가지고 최대 16개의 port를 가질 수 있습니다. 상위 버전으로 스위치가 등장하면서 거의 쓰이지 않습니다.
(Switch:하드웨어를 기반으로 하며 Full-Duplex를 제공하고 몇 백 개의 port를 가질 수도 있습니다.)

*디지털 송수신지

송신지=Source
수신지=Destination

도메인(Domain)

'영역'이라는 뜻입니다.
(브로드캐스트 도메인은 브로드캐스트를 기준으로 나눈 영역을 말합니다.)

Port

통신장비의 interface를 의미합니다. 즉 출입구를 의미합니다.

Link

port와 port를 잇는 경로를 의미합니다.
